# Fgallardograzio Tap

## How do I install these formulae?

`brew install fgallardograzio/tap/<formula>`

Or `brew tap fgallardograzio/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fgallardograzio/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
